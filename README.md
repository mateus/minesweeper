# minesweeper
Minesweeper game made with NodeJS (ES6)

![]()

### Development

Watch SCSS changes:

`npm run build:scss`

Watch JS changes and rebuild JS bundle:

`npm run build:webpack`

Run server with nodemon:

`sudo npm run dev`

### Run

`npm install`

`npm run start`

Go to [http://localhost:8000](http://localhost:8000)
